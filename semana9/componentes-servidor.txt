Sistema operativo:
Es el software base que permite ejecutar Node.js y administrar los recursos del servidor como memoria, red y procesos.

Node.js:
Es el entorno de ejecución que permite correr JavaScript en el servidor para crear aplicaciones web y APIs.

Aplicación server.js:
Es el archivo principal donde se define la lógica del servidor, las rutas y las respuestas a los usuarios.

Puerto:
Es el punto de comunicación por donde el servidor recibe y envía solicitudes, por ejemplo el puerto 3000.

Logs:
Son registros que guardan información sobre la actividad del servidor, errores y eventos importantes para su monitoreo.
